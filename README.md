# WhatsApp Clone

Простой клон WhatsApp с использованием Firebase Firestore для реального времени сообщений.

## Установка

1. Загрузите все файлы в репозиторий GitHub
2. Включите GitHub Pages в настройках репозитория
3. Откройте сайт на телефоне
4. Нажмите "Добавить на главный экран" (Android/iOS)

## Функции

- Реальное время сообщений через Firebase
- PWA - установка на телефон
- Адаптивный дизайн
- Несколько чатов
